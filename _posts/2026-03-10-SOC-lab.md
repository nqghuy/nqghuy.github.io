Mình bắt đầu dựng một lab phòng thủ với các thành phần cơ bản như firewall, SIEM, IDS và EDR. Mục tiêu của lab này là làm quen với cách viết rule cho IDS, EDR, hiểu thêm về attack pattern, học cách đưa log về SIEM và sau đó dựng dashboard để quan sát toàn bộ hệ thống.

Trong phần 1, mình sẽ tập trung cấu hình các thành phần nền tảng của lab.

## 4. Cài Windows client, Sysmon và Splunk UF

Tiếp theo là máy Windows client. Máy này sẽ được cài thêm Sysmon và Splunk Universal Forwarder.

Lý do mình dùng Sysmon là vì:

- Phát hiện các cuộc tấn công không dùng file hiệu quả hơn.
- Giúp truy vết nguồn gốc tấn công thông qua parent process.
- Cung cấp dữ liệu chi tiết và chính xác hơn.
- Hỗ trợ tính hash để phục vụ việc đối chiếu, ví dụ với VirusTotal.

![alt text](/assets/img/posts/SOC_LAB/image-14.png)

Máy Windows đã nhận IP thành công từ DHCP server.

![alt text](/assets/img/posts/SOC_LAB/image-15.png)
![alt text](/assets/img/posts/SOC_LAB/image-16.png)
![alt text](/assets/img/posts/SOC_LAB/image-17.png)

Mình cài Sysmon từ: https://learn.microsoft.com/en-us/sysinternals/downloads/

![alt text](/assets/img/posts/SOC_LAB/image-21.png)

Kiểm tra lại và thấy log Sysmon đã được tạo.

![alt text](/assets/img/posts/SOC_LAB/image-23.png)

Tiếp theo là cài Splunk UF cho Windows:

`wget -O splunkforwarder-10.2.1-c892b66d163d-windows-x64.msi "https://download.splunk.com/products/universalforwarder/releases/10.2.1/windows/splunkforwarder-10.2.1-c892b66d163d-windows-x64.msi"`

![alt text](/assets/img/posts/SOC_LAB/image-22.png)

Sau đó cấu hình file input:

```
[WinEventLog://Security]
disabled = false
index = security_events
sourcetype = WinEventLog:Security

[WinEventLog://Application]
disabled = false
index = windows_system_logs
sourcetype = WinEventLog:Application

[WinEventLog://System]
disabled = false
index = windows_system_logs
sourcetype = WinEventLog:System

[WinEventLog://Microsoft-Windows-Sysmon/Operational]
disabled = false
renderXml = false
index= sysmon
source = XmlWinEventLog:Microsoft-Windows-Sysmon/Operational
```

Lúc này Splunk báo cảnh báo vì chưa có index tương ứng.

![alt text](/assets/img/posts/SOC_LAB/image-24.png)

Mình tạo index mới:

![alt text](/assets/img/posts/SOC_LAB/image-25.png)

Sau khoảng một tiếng loay hoay vì Splunk không nhận log Sysmon, cuối cùng mình tìm ra lỗi cấu hình qua bài viết này: https://www.reddit.com/r/Splunk/comments/1jxjkvo/splunk_not_taking_in_sysmon_source/
